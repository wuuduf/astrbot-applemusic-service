# Code of Conduct

This project adopts the Contributor Covenant Code of Conduct.

## Our Standards

Examples of behavior that contributes to a positive environment:
- Being respectful and inclusive.
- Giving and receiving constructive feedback.
- Focusing on what is best for the community and users.

Examples of unacceptable behavior:
- Harassment, insults, or discrimination.
- Publishing others' private information without consent.
- Any conduct that is inappropriate in a professional community.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards.
They may remove, edit, or reject comments, commits, code, issues, and other
contributions that do not align with this Code of Conduct.

## Scope

This Code applies in all project spaces and public representation contexts.
